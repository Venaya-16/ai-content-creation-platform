# ✍️ AI Content Creation Platform

An AI-powered content generation platform built using **Google Gemini** that helps users create blogs, social media captions, ad copies, and SEO-friendly content with just a few inputs.

---

## 📖 Project Overview

Creating high-quality content for different platforms often requires significant time and effort. Whether it's writing a blog, crafting an engaging social media caption, creating persuasive advertisements, or optimizing content for search engines, AI can significantly streamline the process.

This project leverages **Google Gemini** to automate multi-format content creation while allowing users to customize the audience and writing tone.

---

## 🚀 Features

- 📝 Generate SEO-friendly blog articles
- 📱 Create engaging social media captions
- 📢 Generate persuasive ad copies
- 🔍 Produce SEO-optimized titles, meta descriptions, keywords, and blog outlines
- 🎯 Customize content for different audiences and writing tones
- 🤖 Powered by Google Gemini
- 🖥 Interactive Gradio interface

---

## 🛠 Tech Stack

- Python
- Google Gemini API (Gemini 2.5 Flash)
- Google Gen AI SDK
- Gradio
- Google Colab
- Prompt Engineering

## ⚙ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/ai-content-creation-platform.git
```

Move into the project folder

```bash
cd ai-content-creation-platform
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a Google Colab Secret

```
GEMINI_API_KEY
```

Run all notebook cells.
